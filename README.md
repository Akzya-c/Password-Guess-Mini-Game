# Password Cracking Puzzle (Educational)

This is a simple educational Python puzzle game where the player attempts to guess a hidden password using logical hints.  
The project is designed for learning purposes only and does not involve real password cracking or security vulnerabilities.

The goal is to help beginners understand string comparison, loops, and conditional logic in Python.

---

## How the Puzzle Works

1. A predefined password is stored in the program.
2. The player is given a hint about the password length.
3. The player guesses the password within a limited number of attempts.
4. After each incorrect guess, the program displays:
   - The number of correct letters in the correct position.
5. The game ends when the password is guessed correctly or attempts are exhausted.

---

## Technologies Used

- Python 3
- Basic string operations
- Loops and conditionals
